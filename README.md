# Market App

## Project Description

Market App is a simple web-based application developed using Python and HTML templates.
The goal of the project is to demonstrate basic concepts of web development, including routing, working with forms, and database interaction.

The application allows users to interact with market items through a web interface while storing the data locally.

---

## How to Run the Program

1. Make sure Python 3 is installed on your computer.

2. Clone the repository:

```bash
git clone https://github.com/Kirill-ark/Market-App.git
cd Market-App
```

3. Install dependencies:

```bash
pip install flask
```

4. Initialize the database:

```bash
python init_db.py
```

5. Run the application:

```bash
python app.py
```

6. Open your browser and go to:

```
http://127.0.0.1:5000
```

---

## Features Implemented

* Web interface using HTML templates
* Displaying market items
* Adding and managing data through forms
* Navigation between pages
* Database initialization script
* Backend logic implemented in Python

---

## How Data is Stored

The application uses a local database created during initialization.

* The database is generated by running the `init_db.py` script
* Data is stored locally on the user's machine
* The project most likely uses SQLite, which does not require a separate server
